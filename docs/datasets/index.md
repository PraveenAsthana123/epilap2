## Data Governance Note

> **Why:** Every dataset carries sensitive PHI/genomic data. **How:** All dossiers frame the
> AI as **decision support, never autonomous diagnosis, prescribing, or surgical decisions**,
> and require consent, de-identification for research, and human (Neurologist) oversight.
